# ilmansuunnat

Pohjoista ja etelää lukuunottamatta minulla on taipumus sotkea ilmansuunnat. Päätin siis tehdä aiheesta pelin, jolla harjoitella.

Käynnistys:
```
# python ilmansuunnat-pygame.py
```
